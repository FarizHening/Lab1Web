### Pemformatan Text 

Pemformatan text dilakukan untuk memodifikasi tampilan text pada browser supaya tampilan text lebih menarik.

Untuk kode yang dipakai sebagai berikut :

`<b>` : Bold text

`<strong>` : Important text

`<i>` :Italic text

`<em>` :Emphasized text

`<mark>` :Marked text

`<small>` :Smaller text

`<del>` :Deleted text

`<ins>` :Inserted text

`<sub>` :Subscript text

`<sup>` :Superscript text
